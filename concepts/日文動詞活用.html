日文動詞活用（Japanese Verb Conjugation）

日文動詞的核心特性：所有動詞原形（辭書形）的最後一個音必定是う段音。依據最後兩個音的特徵，可判斷動詞類型，進而套用對應的活用口訣。

一、動詞五類別

類別傳統名稱現代（みんなの日本語）判斷特徵
五段動詞五段活用第一類（Group 1）語尾非る，或語尾為る但前音不是い/え段
上一段動詞上一段活用第二類（Group 2）語尾：い段音＋る
下一段動詞下一段活用第二類（Group 2）語尾：え段音＋る
するサ行變格活用（サ変）第三類（Group 3）する 本身及「漢語名詞＋する」
くるカ行變格活用（カ変）第三類（Group 3）来る 唯一一個

核心重點

する 佔日文動詞的 90% 以上，任何含有動作意義的漢語名詞都可加 する 轉成動詞。

二、動詞類別判定四步驟（§6）

步驟1：先排除兩個不規則動詞
       する → サ変（第一變化有三種：さ/し/せ）
       来る → カ変（讀音在き/く/こ之間變化）

步驟2：語尾不是る → 一定是五段動詞
       （語尾是う、く、ぐ、す、つ、ぬ、ぶ、む）

步驟3：漢字＋い段音＋る → 上一段動詞
       漢字＋え段音＋る → 下一段動詞

步驟4：漢字＋単一る（前音是い/え段音的漢字讀音）
       → 原則先當五段動詞，除了以下例外：

  上一段（9個）：居る・見る・似る・出来る・着る・煮る・干る・射る・鋳る
  下一段（4個）：寝る・出る・経る・得る
  其餘 → 五段動詞

📖 老師怎麼說：る結尾的動詞為什麼最麻煩？

所有動詞的原形最後一個音一定是う段音。麻煩的不是其他語尾，而是 る——因為 る 可能是五段、上一段、或下一段，要靠倒數第二個音判斷。

判別邏輯（四步走）：

  1. 先確認是不是 する 或 来る（不規則），是的話不用再判斷。
  2. 語尾不是 る → 直接是五段（思う、書く、話す 之類）。
  3. 漢字後面接兩個假名、倒數第二個是い/え段＋る → 一定是上/下一段（生きる、食べる）。
  4. 漢字後面只接一個 る → 「先當五段」，只有 9＋4＝13 個例外要背。

老師的建議：遇到「漢字＋る」的字，先把它當五段，讀到那個字出現第一變化的時候，如果不對就修正。9個上一段（居る・見る・似る・出来る・着る・煮る・干る・射る・鋳る）和4個下一段（寝る・出る・経る・得る）是真正要背的。

三、六種活用形（變化形）

變化形號碼傳統名稱功能後接範例
否定形①未然形（否定）否定、被動、使役ない、ず、られる、させる
推量形①’未然形（推量）推量/勸誘よう（上下一段）、よう→お段（五段第⑦形）
中止形②連用形連接另一動作、加助動詞ます、た、たい、て、ながら
終止形③終止形句子終結、辭書形句點、と、から、が、か、そうだ（傳聞）
連體形④連体形修飾名詞（修飾語末尾）名詞、ので、のに、ようだ
假設形⑤仮定形假設語氣ば
命令形⑥命令形命令（句點）

四、上/下一段動詞活用（§9）

口訣：X、X、る、る、れ、ろ（只有語尾る在變）

變化見る（上一段）出る（下一段）後接
① 否定見（る去掉）出＋ない、られる、させる、よう
② 中止見出＋ます、て、た、たい、ながら
③ 終止見る出る句點、から、と、か
④ 連體見る出る＋名詞、ので、のに
⑤ 假設見れ出れ＋ば
⑥ 命令見ろ出ろ（句點）

①②的實現：把る去掉即可（兩者形式相同，只靠後接的助動詞/助詞分辨）

五、五段動詞活用（§11）

口訣：a-i-u-u-e-e-o（最後的う段音在五段上變化，第3/4重複，第2/5重複）

六、五段動詞音便（§12）

五段動詞②形加 た・て・たり 時發生音便（語音同化）：

音便類型觸發行變化例
促音変あ行う、た行つ、ら行る②→促音（っ）思う→思った、待つ→待った、帰る→帰った
い音変か行く②→い書く→書いた、聞く→聞いて
鼻音変な行ぬ、ば行ぶ、ま行む②→ん（後続濁音化）読む→読んだ、遊ぶ→遊んで、死ぬ→死んだ
無音変さ行す②=し（不变）話す→話した（直接接続，無音変）

唯一例外

行く（か行）→ 照規則應為い音変，但行く 做促音変：行った、行って

還原技巧（閱讀時逆推原形）

  • 看到 ん＋だ/で → 鼻音変（読んで → 読む）
  • 看到 促音っ＋た/て → 促音変（思った → 思う）
  • 看到 い＋た/て → い音変（書いた → 書く）

七、する 活用（§16）★★★

する 整個字都在變（無語幹），90% 以上的日文動詞是「漢語名詞＋する」形式。

變化形形式範例後接
① 否定さしない・されないない、れる、させる
①’ 推量ししようよう、まい
① 文語否定せせず・せられるず、られる、しめる
② 中止しします・した・してます、た、て、たい、ながら
③ 終止するする句點、と
④ 連體するする＋名詞名詞、ので
⑤ 假設すれすればば
⑥ 命令しろ/せよしろ/せよ（少見）

佛典論文高頻句型

論文常見 する 變化形：

  • します / した / している / していた（ています 狀態）
  • することができる（能夠做某事）
  • することになる（變成要做某事）
  • するという（内容限定，「說…」）

ずる 變體（佛典論文常見）：在鼻音後，する 變成 ずる（す→ず 濁音化）

  • 信ずる（しんずる）、論ずる（ろんずる）、生ずる（しょうずる）
  • ①形：ざ/じ/ぜ；②形：じ；③④形：ずる；⑤形：ずれ

八、くる 活用（§18）

来る 的讀音在 き/く/こ 之間變化（カ行變格活用）：

變化形形式（漢字）讀音後接
① 否定来こない（来ない）、られる（来られる）、させる（来させる）
② 中止来きます（来ます）、た（来た）、て（来て）、たい（来たい）
③ 終止来るくる句點、と
④ 連體来るくる名詞
⑤ 假設来れくれば（来れば）
⑥ 命令来いこい春よ、来い！

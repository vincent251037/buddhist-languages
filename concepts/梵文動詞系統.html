梵文動詞系統（Sanskrit Verb System）

梵語動詞結構：字根（root）＋後綴（suffix）→ 語幹（stem）＋語尾（ending）

影響語尾變化的四個因素：

因素選項
人稱第1、第2、第3 人稱
數量單數、雙數、複數
語態主動（為他 parasmaipada）、被動/中動（為己 ātmanepada）
語氣直述、命令、祈願

梵語動詞不需要代名詞主語——一個動詞本身就是完整句子。

一、母音強化階次（guṇa / vṛddhi）

梵語字根母音有三個強化層級，影響語幹的構成：

階次說明例（i/u/ṛ）
零次（zero grade）字根本身，不強化i, u, ṛ
guṇa（一次強化）短 a 強化e（←i）, o（←u）, ar/al（←ṛ）
vṛddhi（二次強化）長 ā 強化ai（←i）, au（←u）, ār（←ṛ）
√budh → guṇa → bodh（知道）
√bhū  → guṇa → bho → bhav（存在）

二、現在系統（Present System）

包含四種時態/語氣，全部使用現在語幹：

  1. 現在式（Present）
  2. 未完成式（Imperfect）
  3. 祈願式（Optative）
  4. 命令式（Imperative）

現在式語尾（一次語尾）

以 √hṛ（拿）為例：

人稱單數雙數複數
第3人稱haratiharataḥharanti
第2人稱harasiharathaḥharatha
第1人稱harāmiharāvaḥharāmaḥ

⚠️ 第一人稱：語幹末短 a → 長 ā（接 m/v 開頭的語尾時）

三、十組動詞（現在語幹構成）

主題性動詞（Thematic，1/4/6/10 組）

共同特徵：現在語幹以短 -a 結尾

組別語幹構成法例
第1組字根 guṇa 強化 ＋ a√budh → bodh+a → bodhati
第4組字根 ＋ ya（不強化）√nṛt → nṛt+ya → nṛtyati
第6組字根 ＋ a（不強化）√tud → tud+a → tudati
第10組字根 guṇa 強化 ＋ aya√cint → cint+aya → cintayati

第10組特殊規則：

  • 構成未來式時，用現在語幹去短 a（非字根）+ iṣya
  • √cint → cintay(a) + iṣya → cintayiṣyati（不是 cintiṣyati）

不強化的三種情況（第1、10組）：

  1. 字根母音已是短 a 或長 ā → 不需強化
  2. 字根末為長母音或複雜母音後接子音 → 不強化（如 √jīv → jīvati）
  3. 字根末為短母音加複合子音 → 不強化（如 √cint → cintayati）

常見不規則：

√gam → gacchati（非 gamati）
√dṛś → paśyati（非 dṛśati）
√bhū → bhavati（o → av 內連音：bho+a → bhava）
√ji  → jayati （e → ay 內連音：je+a → jaya）

非主題性動詞（Athematic，2/3/5/7/8/9 組）

  • 現在語幹不以短 a 結尾
  • 有強語幹與弱語幹之別
  • 為他動詞（P）：單數用強語幹，雙數/複數用弱語幹
  • 為己動詞（Ā）：全用弱語幹

四、過去時態

梵語有三種過去時態 + 一種佛典慣用法：

時態構成用途
未完成式 Imperfect前增 a ＋ 現在語幹 ＋ 二次語尾較近的過去；敘述遠的過去（古典梵語）
完成式 Perfect重疊語幹說話者未目睹的遙遠過去
Aorist多種構成方式無特定時間限制的過去
現在式 ＋ sma現在語幹 ＋ sma 後置詞佛典最常用：表示已發生的過去
未完成式：a + vada + t → avadat（他說了）
佛典用法：bodhisattvaḥ śramaṇam paśyati sma
         菩薩看到了沙門（sma 標記為過去發生的事件）

sma 有無的差異：

  • 有 sma → 動作在那一時刻完成（點狀）
  • 無 sma → 現在式敘述過去，動作持續一段時間

過去被動分詞（ppp）代替過去式（佛典極常見）：

rājā mṛtaḥ    → 國王死了（mṛta = ppp. of √mṛ）
Yamaḥ prayātaḥ → Yama 出發了

五、命令式（Imperative）

使用時機：

語境人稱例
命令/請求第2人稱gaccha! 你去！
願/讓第3人稱bhavatu 讓它是 / 願它是
願望第1人稱願我⋯⋯

構成：在語尾加 -tu（第3人稱單數）等命令語尾

否定命令：用 mā（禁止），不用一般的否定詞 na

mā gaccha.  ← 不要走！（命令否定）
na gacchati ← 他不去（直述否定）

六、祈願式（Optative / Potential）

不是時態，是語氣——表達「動作還沒發生」的三種情況：

語氣中文對應例
義務/責任should, ought to, must應當去
可能性could, might, may可能去
假設/意願would假設會去

構成：現在語幹 ＋ ī/yā ＋ 二次語尾

識別標記：主題性動詞的祈願式語幹末尾出現 e 音（a + ī → e）

七、被動語態（Passive Voice）§164

構成：字根 ＋ ya ＋ 為己（ātmanepada）語尾

被動句的主語用主格；動作發出者用具格：

rāmeṇa  buddhaḥ  dṛśyate
 (具格)   (主格)    被看見
→ 佛陀被 Rāma 看見

ya 形式辨析（四種 ya 用法容易混淆）：

形式構成方式意義
被動語態字根 + ya + 為己語尾被⋯⋯
連續體接頭詞 + 字根 + ya做完A（再做B）
未來被動分詞 fpp字根 + (i)ya + 名詞語尾應當被⋯⋯的
使役動詞現在語幹 + aya + 動詞語尾使/令某人做⋯⋯

八、使役動詞（Causative）§159

構成：現在語幹 ＋ aya ＋ 現在式語尾

使役的對象（令誰去做）用具格：

ācāryaḥ śiṣyam pathaṃ gacchayati
老師(主格) 讓學生(受格) 去路上
→ 老師讓學生走這條路

九、未來式（Future Tense）

構成：字根 ＋ sya / iṣya ＋ 一次語尾

√gam + iṣya → gamiṣyati（他將去）
√ji + sya → jeṣya（guṇa: i→e；內連音: s→ṣ）

特殊規則：簡單母音（i/u/ṛ）字根 + sya 時，需 guṇa 強化

第10組動詞例外：不用字根，用現在語幹去短 a + iṣya

十、名詞性句構（Nominal Sentences）

「A 是 B」句型，繫詞 bhavati 常省略：

sarvadharmāḥ śūnyāḥ (bhavanti)
一切諸法  是空的（複數 bhavanti 省略）

→ 省略時主謂可互換：
śūnyāḥ sarvadharmaḥ  ← 謂語前置，強調「空」

判斷主語/謂語：通常「已知信息」是主語，「新信息」是謂語

存在句：asti（有 / 是）、bhavati（存在 / 成為）、vidyate（存在，為己語態）

十一、動詞語根名詞化

字根 guṇa 強化後 ＋ a 語基 / i → 名詞：

√budh + i → bodhi（覺悟、菩提）

My Position

sma 用法對讀早期佛典至關重要——《阿含經》類的佛典大量使用「動詞現在式 + sma」表達過去的法會情境，若不識別這個標記，容易誤以為是描述當下發生的事。

被動語態在佛典論書中極為常見（如：「…解釋為…」、「…被稱為…」），識別「字根 + ya + 為己語尾」是閱讀論書的基本功。

第十組動詞 + aya（使役形）與被動的字根 + ya外型近似，需透過語尾（一次語尾 vs. 為己語尾）區分。

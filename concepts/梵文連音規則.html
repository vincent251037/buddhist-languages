梵文連音（Sanskrit Sandhi）

Sandhi（संधि）是梵語為確保音律一致性而整理出的系統性音變規則：當相鄰音節或詞語接觸時，為發音便利而產生的語音同化。

梵語是唯一將口語音變完整書面化的古典語言。這意味著讀梵文時，你看到的文字已是連音後的結果——必須學會「逆向還原」，才能辨認詞界。

一、兩大原則

類型原則目的
子音連音相鄰子音發聲方式趨同（有聲/無聲同化）消除發音障礙
母音連音相鄰母音合併（去除母音間隙）讓詞與詞之間流暢連接

二、內連音（Internal Sandhi）

詞語內部字母間的固定音變規則，影響詞形本身。

2a. n → ṇ（反舌化）§15

觸發條件：同一詞中，n 前面有 r、ṣ、ṛ 且 n 後面接母音、m、v、y 或另一個 n。

sūtrani  → sūtrāṇi    （複數位格，n 受 r 影響反舌化）
nṛpena   → nṛpeṇa     （具格，n 受 ṛ 影響）
brahman  → brahman     （n 後面為空白，不變）
brahmaṇā              （加格位後綴，n 後接母音 a → 反舌）

2b. s → ṣ（反舌化）§32

觸發條件：同一詞中，s 前面緊接著短 a / 長 ā 以外的母音。

naresu  → nareṣu    （e 前面的 s → ṣ）
mokṣaḥ             （已是反舌 ṣ 的結果）
niṣīdati           （ni + sīdati）
tiṣṭhati           （ti + sthati：s 反舌後，連帶 t 也反舌→ ṭ）

2c. c/ch/j/jh/ś → k（接 s 時）

字根末尾為這五個音接 s（未來式語幹 sya）時：

√muc + sya → mukṣya   （c → k；k前的s → ṣ）→ 強化 → mokṣya
√dṛś + sya → dṛkṣya   （ś → k）→ drakṣya（不規則）
√budh + sya → butsya  （有聲有氣音 dh → 無聲無氣音 t）→ bhotsya

三、外連音（External Sandhi）

詞與詞之間的音變，是閱讀梵文最常遇到的障礙。

3a. 子音外連音大原則

無聲子音（如 t）：
  + 無聲子音  → 不變（例外：遇硬顎/反舌噝噝聲則逆向同化）
  + 有聲子音  → 變有聲

有聲子音（如 n）：
  + 無聲子音  → 不變（例外：遇硬顎/反舌噝噝聲則有特殊噝聲變化）
  + 有聲子音  → 不變（例外：遇有聲硬顎/反舌）

3b. Anusvāra 連音（ṃ / म्）

字尾 m + 下一字開頭為子音 → m 變 anusvāra（ṃ）

sam + gha  → saṃgha（僧伽）
              有時因 gh 進一步同化 → saṅgha

taṃ aśvaḥ  → tamaśvaḥ   （下一字開頭為母音 a，m 保持不變）
⋯⋯ rāmaṃ.  → ⋯⋯ rāmam. （句尾，anusvāra 不用）

Anusvāra 不出現於：母音之前、句尾。

3c. Visarga 連音（ḥ / ः）

Visarga 是梵文最複雜的連音，分三大規則：

規則 1：-aḥ/-āḥ 結尾

後接音變化例
短 a 以外的母音ḥ 脫落naraḥ upagacchati → nara upagacchati
短 aaḥ→o；下字短a→‘naraḥ avagacchati → naro ‘vagacchati
有聲子音aḥ→o；āḥ→長ā保持naraḥ gacchati → naro gacchati；narāḥ gacchati → narā gacchati
無聲子音ḥ 保持（無變化）—

規則 2：非 a/ā 結尾的 -ḥ

後接音變化例
母音或有聲子音ḥ → rnaraiḥ gajam → narair gajam
r-ḥ 消失；前母音變長bhikṣuḥ rakṣati → bhikṣū rakṣati
無聲子音同 -aḥ/-āḥ 規則—

規則 3：saḥ/esaḥ（第三人稱代名詞）例外

saḥ + 任何子音/非短a母音 → sa（ḥ 直接脫落）
saḥ gacchati → sa gacchati

saḥ + 短 a → so 'tra（同 aḥ 的短a規則）
saḥ atra → so 'tra

e/o 碰到短 a：特殊省略

vane atra → vane 'tra   （e/o 保持不變，後字短a省略為 '）

3d. 母音外連音（Vowel Sandhi）

情況變化例
簡單母音 + 相同母音合併為同一長母音gacchāmi iti → gacchāmīti
i/ī + 異種母音i/ī → yiti ācāryaḥ → ity ācāryaḥ
u/ū + 異種母音u/ū → vna tu atra → na tv atra
ṛ/ṝ + 異種母音ṛ/ṝ → r—
a/ā + i/ī→ eatra iti → atreti
a/ā + u/ū→ ona uttiṣṭhati → nottiṣṭhati
a/ā + ṛ→ arsenā ṛsiḥ → senarsiḥ
e + 短ae 不變，短a→ ‘vane atra → vane ‘tra
e/o + 非短a母音e/o → 短avane ācāryaḥ → vana ācāryaḥ
ai + 任何母音ai → ā—
au + 任何母音au → āvgajau atra → gajāv atra

⚠️ 例外：呼格（Vocative）和感嘆詞不做母音連音變化。

四、逆向還原技巧（閱讀時）

讀梵文佛典時，需從連音後的形式還原詞界：

看到 narair → 原形是 naraiḥ（具格複數）
看到 naro 'tra → 原形是 naraḥ + atra
看到 ity → 原形是 iti（iti 的連音）
看到 saṃgha → 原形是 sam + gha（僧伽）
看到 tiṣṭhati → 原形是 sthā（動詞語根）的現在式
看到 mokṣaḥ → 解脫（含有內連音 ṣ）

重要識別線索：

  • 看到 ‘（無字母符號）→ 前面有 aḥ/e/o 的連音省略了短 a
  • 看到 ṃ → 前面詞尾原本是 m
  • 看到 r 夾在母音中 → 可能是 visarga 的 ḥ→r 變化
  • 看到 ṇ / ṣ / ṭ → 反舌化，找前面的 r/ṣ/ṛ

My Position

連音是「梵文難以入門」最大原因之一：一句話在連音後看起來與原詞形完全不同。還原策略比記憶規則更重要——應培養「看到異形就問這可能是什麼的連音結果？」的反射。

佛典（特別是詩偈）的連音密度極高；散文佛典（如《大般若經》）有時刻意「少連音」以便閱讀——這也是判斷文本風格的線索。

日文音便（S4 附記）與梵文 Sandhi 在原理上相通（鄰音同化），但梵文規則更系統、更強制；日文音便僅適用於有限情境（第二變化加た/て）。

日文助動詞（Japanese Auxiliary Verbs）

日文助動詞的核心規則：助動詞長得像哪種詞類就按那種詞類變化。

一、助動詞分類總表（§39）

變化類型助動詞功能
像下一段動詞（ら）れる、（さ）せる、しめる被動、使役
像五段動詞たがる、である希望（他稱）、Be動詞（文章體）
像い形容詞ない、たい、らしい否定、希望、推測
像な形容詞そうだ（樣態）、そうだ（傳聞）、ようだ樣態、傳聞、樣態
特殊變化形ます、た、です、ず、ざり美化、過去、Be動詞（敬體）、文言否定
無變化まい、（よ）う、ん否定推量、推量/勸誘、文言否定

📖 老師怎麼說：助動詞不用死背，看長相就知道怎麼變

助動詞的變化原則只有一條：長得像哪種詞類，就按那種詞類的口訣變化。

例如 れる（被動），最後一個音是 る，倒數第二個音是 れ（え段音）→ 按下一段動詞口訣（X X る る れ ろ）變化。

たい（希望），最後一個音是 い → 按 い 形容詞口訣變化，所以否定是 たくない，過去是 たかった，不是 たなかった。

這個邏輯很重要：你不需要另外背一套助動詞活用表，只需要判斷它「長得像誰」，再套用你已經學過的口訣。

二、被動・使役（§40）

被動（ら）れる

動詞類型被動形
五段動詞V① ＋ れる
上下一段動詞V① ＋ られる
するされる
くるこられる

被動形本身像下一段動詞 → 加其他助動詞時，把れる/られる當下一段動詞處理。

使役（さ）せる

動詞類型使役形
五段動詞V① ＋ せる
上下一段動詞V① ＋ させる

複合範例

書く① + せる（使役）① + られる（被動）① + ない（否定）② + た（過去）① + う（推量）

書か - せ - られ - なかっ - たろ - う

→ 書かせられなかったろう。
   昨天沒有被（強制）要求寫吧！

還原技巧：從後往前讀——う→たろ（推量→過去推量）→なかった（否定過去）→られ（被動）→せ（使役）→書か（書く の①変化）

三、様態・傳聞・推量（§43）★★★

そうだ 兩種截然不同的意義

前接意義辨別
V②（中止形）＋そうだ様態（好像、似乎）動詞用②變
V③（終止形）＋そうだ傳聞（據說、聽說）動詞用③變
い形容詞語幹＋そうだ様態去掉い
な形容詞語幹＋そうだ様態直接用語幹
見そうだ  ← 見る②中止形 → 好像要看
見るそうだ ← 見る③終止形 → 據說要看

ようだ / らしい

助動詞前接意義
ようだV④（連體形）＋ようだ様態（好像，根據客觀觀察）
らしいV③（終止形）＋らしい推測（好像，根據情報推斷）

四、ます・た 特殊變化（§44）

ます（美化/禮貌語體助動詞）

前接：動詞②（中止形）

変化形形式用例
①ませ/ましょません（否定）；ましょう（勸誘：Let’s…）
②ましました（過去）；まして（中止）
③④ます終止/連體
⑤ますれますれば
⑥ませいらっしゃいませ（歡迎光臨）

た（過去助動詞）

前接：動詞・助動詞の②

⚠️ た は②変化を持たない（た前は必ず②変化；た自身も変化する）

た の形式変化形用例
たろ①たろう（過去推量：…だったろう）
（なし）②—
た③④た（句點結束/連體修飾）
たら⑤たら（假設語氣，後不加ば！）

常見複合句型：なければならない

なければならない = ①＋なければ + ならない
（不這樣做不行 = 一定要做）

ない の⑤変化 = なけれ（因為ない長得像い形容詞：い→けれ）
ならない = なる①＋ない（不成/不行）

文言形：①＋ねばならぬ（同義，偶見於學術文章）

五、だ・です・である（§45）★★★

Be動詞的三種文體，論文閱讀最重要。

形式文體出現場合
です敬體口語、對話
だ常體論文、學術文章（主要）
である文章體論文、學術文章（主要）

だ の変化（論文必備）

変化形形式用法
①だろだろう（推量：…吧）
②で（中止）/ だっ（過去）で：中間停止形 ★★★；だった：過去
③だ終止（句尾）
④な連體修飾（な形容詞語法）
⑤なら假設語氣（なら可以…）

最重要：だ の②変化「で」—— 中止形 ★★★

阿含経とは、最も古い仏教経典集であり、釈迦の言葉を色濃く反映した...
阿含**とは** = 所謂...
**であり** = である（文章體Be動詞）の②変化（中止形）= 是...而且...

阿含とは、サンスクリット・パーリ語のアーガマの音写**で**、
「伝承された教説」という意味**である**。
この**で** = だ の②変化（中止形）= 是...（不是格助詞で！）

辨別 で 是格助詞還是 Be 動詞②変化：

判斷で の性質翻譯
名詞＋で＋動詞（方法手段）格助詞用…
名詞（=是）＋で、続くだ の②変化是…（而且）

口訣：「學生で」翻成「用學生」或「在學生」都很奇怪 → 那就是 Be 動詞 だ の②変化「是學生…」

である の変化（文章體）

である = で（だ②）+ ある（存在動詞）
→ 按 ら行五段動詞変化：ら・り（っ）・る・る・れ・れ・ろ

②であり：中止形（A はBであり、CはDである ＝ A是B且C是D）
②であった：過去（は학생であった ＝ 是學生的）
③④である：終止・連體
⑤であれ（ば）：假設
否定：では（は）ない

論文範例：

AはBであり、CはDである。  A是B，C是D。（用であり並列）
AはBとCである。          A是B和C。
AはBであった。           A曾是B（過去式）
AはBではない。           A不是B（否定）

です の主要変化

変化形用例
でした過去（私は学生でした）
でしょう推量（…でしょう = …吧！）
ではありません否定（私は学生ではありません）

六、文言助動詞（§46）

佛典學術文章偶見：

文言形現代對應意義常見変化
なし（無し）ない不存在、無④なき（最常見）
ごとし（如し）～のように如同…一樣V④が＋ごとし；名詞の＋ごとし
べし（可し）べきだ應該、能夠V③＋べし

七、下樓梯模型（§47）：助動詞連接順序

助動詞必須依照特定順序疊加，從最靠近動詞到最末尾：

動詞
  ① 使役（させる・せる）
  ② 被動（られる・れる）
  ③ 否定（ない）
  ④ 過去（た）
  ⑤ 推量（う・ろう）

書かせられなかったろう。
書く ← せる ← られる ← ない ← た ← う
    使役   被動   否定   過去  推量
→ 昨天大概沒有被（強制）要求寫吧！

還原方法：從句點往前，一層一層剝開助動詞：

  • 最後的推量詞 → 過去助動詞 → 否定 → 被動 → 使役 → 動詞原形

📖 老師怎麼說：用下樓梯讀懂複合助動詞

下樓梯模型是讀日文論文最實用的工具。遇到一長串助動詞疊加時，先不要慌——從句尾往前剝，像下樓梯一樣一層一層：最外層是語氣（推量/過去），中間是否定，再進去是被動/使役，最核心是動詞。

老師的練習題：「書かせられなかったろう」——

  • ろう = 推量
  • なかった = 否定過去（ない的連用形＋た）
  • られ = 被動
  • かせ = 使役
  • 書く = 動詞原形

回推：寫（書く）→ 被要求（させる）→ 被迫（られる）→ 沒有（ない）→ 過去（た）→ 大概（う）。整句意思：「大概昨天沒有被強迫寫吧！」

たい（希望）是最常見的例子：食べる → 食べたい（想吃）→ 食べたくない（不想吃）。注意 たい 按 い 形容詞變，否定是 たくない，不是 たない。

My Position

で（だの②変化） 是閱讀佛典日文最關鍵的識別點。不識別這個 で，定義句的翻譯會完全出錯。凡遇到「名詞で、…」或「名詞で…という」的結構，優先判斷是 Be動詞（是…），而非格助詞（用…/在…）。

であり / であった 幾乎出現在每一頁佛學論文中。であり 是中止形，用來並列兩個判斷句；掌握この二形即可讀懂大多數佛學論文的論點陳述句式。

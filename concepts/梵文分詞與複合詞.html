梵文分詞、代名詞與複合詞（Sanskrit Participles, Pronouns & Compounds）

S7 涵蓋梵文閱讀最需要的三個工具：分詞（把動詞功能轉入名詞/形容詞）、代名詞（句子的指稱系統）、複合詞（六離合釋，佛典術語的基本構成方式）。

一、連續體（Absolutive / Gerund）§103-104

定義

連續體是不變化詞（indeclinable）——不隨人稱、數量、時態變化。意義上表示「在…之後 / 已做了…之後」（= After V-ing / Having done…）。

三個核心限制：

  1. 語序決定動作先後，不可隨意調換
  2. 連續體與主句動詞的主語必須相同
  3. 連續體可有自己的受詞

構成

情形構成方式例
無前綴字根字根 ＋ tvā（或 itvā）√ji → jitvā（征服之後）
有前綴字根前綴 ＋ 字根 ＋ yapra√viś → praviśya（進入之後）
字根以短母音結尾前綴 ＋ 字根 ＋ tyavi√ji → vijitya

例句

sarvair janair militvā  sa  nṛpo  vṛtaḥ
全部民眾  集合之後  那個 國王  被選出（ppp）

→ 全部民眾集合之後，那個國王被（所有民眾）選出。
  （「集合」和「選出」的主語相同：全部民眾）

與獨立位格的區別：

  • 連續體 → 同一主語的前後動作
  • 獨立位格 → 不同主語的從屬子句

二、不定體（Infinitive）§111

功能：表達目的（= 英文 “to V”），永遠不變化。

構成：字根 guṇa 強化 ＋ -tum（或 -itum）

翻譯意涵（視語境選擇）：

  1. 為了⋯⋯
  2. 能夠⋯⋯
  3. 想要⋯⋯
  4. 打算⋯⋯
  5. 合適⋯⋯

三、代名詞系統

代名詞無呼格；性取決於所代替或修飾的名詞。

人稱代名詞

人稱主格語根形式
第1人稱aham（我）mad / asmad
第2人稱tvam（你）tvam / yuṣmad
第3人稱saḥ/sā/tat（他/她/它）tad

簡略型（me, te, nau 等）：不能置於句首，不能與 eva/ca/vā 連用。

tad 的三種用法

tad（第三人稱）是梵文最多功能的代名詞：

用法說明例
人稱代名詞he/she/itsaḥ gacchati（他去）
指示代名詞that/thosetat paśyati（他看那個）
指示形容詞that + 名詞saḥ bālaḥ（那個男孩）

辨別技巧：若句中已有另一個主格名詞，則 saḥ 是指示形容詞（the/that），不是代名詞 he。

指示代名詞距離系統

由近而遠：etad（此，this）→ idam（此）→ tad（彼，that）→ adas（彼，遠指）

佛典定型句：etad abhavat = 「作是念」（他這樣想到）

疑問代名詞（§70）

梵文疑問詞以 k- 開頭（對應英文 wh-）：

形式意義
kaḥ / kā / kimwho / what
kutrawhere
kadāwhen
katamawhich（佛典常用，尤其用於列舉式問答）

不定代名詞：疑問代名詞 ＋ 後綴 cit / cana / api → 「某某、任何一個」

四、分詞系統

4a. 過去被動分詞（PPP）§60 ★★★

構成：字根（弱語幹）＋ (i)-ta（多數）或 -na（少數，字根末為母音/g/j/d 時）

√kṛ + ta → kṛta（被做的）
sam + kṛta → saṃskṛta（梵語！ = 被一起做好的 = 精緻化的）
√budh + ta → buddha（覺悟的）
√gam + ta → gata（已去，主動！）

語態辨別：

  • 及物動詞字根 → 通常被動（被⋯⋯的）+ 有過去意涵
  • 不及物動詞字根 → 主動（已⋯⋯的），無被動意涵
gata → 已去的（非「被去」）
mṛta → 已死的（非「被死」）

PPP 四種用法：

用法說明位置
1. 屬性形容詞修飾名詞（被⋯⋯的）名詞前
2. 情況形容詞功能像副詞，表時間/地點（在⋯⋯之後）靠近動詞
3. 名詞化直接當名詞用—
4. 取代有限動詞代替過去式動詞（預設第3人稱）—

以 PPP 取代被動動詞（例句）：

taṃ buddhena nirdiṣṭaṃ mārgam anugacchāmaḥ
那個 被佛陀(具格)  指出的(ppp)  道  我們隨行

→ 我們尋著被佛陀教導的那條道。
  （nirdiṣṭa 的性數格跟隨修飾對象 mārgam：m.sg.acc.）

4b. 過去主動分詞（§198）

PPP ＋ vat（vant） → 主動語態的過去分詞，甚至可完全取代有限動詞

4c. 未來被動分詞（FPP）§109

表達應當或可能被做某事（= should be done / can be done）：

構成：字根 ＋ (i)ya、(i)tavya、(i)anīya ＋ 名詞語尾

4d. 現在分詞（§142/§168）

  • 主動：字根 ＋ 現在語幹 ＋ -at（為他）/ -āna（為己）
  • 被動：被動語幹 ＋ māna（being done）
  • 注意：現在分詞不能取代有限動詞（不同於 PPP）

佛典範例：

√as（存在）→ s + at = sat（存在的）→ 正確的、善的
（妙法蓮華經之「妙法」= 善法/正法）

五、複合詞（六離合釋）ṣaṭ-samāsāḥ

梵語複合詞是佛典術語最重要的構成方式。三個基本規則：

  1. 前分不做格位變化（用原形/弱語幹）
  2. 性別取決於後分
  3. 詞間產生外部連音

六種複合詞類型

梵文名傳統譯名中文說明英文
karmadhāraya持業釋前後同格（A 即是 B）Appositional
tatpuruṣa依主釋前分限定後分（A 的 B / B 關於 A）Determinative
bahuvrīhi有財釋前後合指另一名詞（具有 A-B 的）Possessive
dvandva相違釋並列（A 和 B）Copulative
avyayībhāva不變化釋前分為副詞，整體不變化Adverbial
dvigu帶數釋前分為數詞Numeral

持業釋（karmadhāraya）——前後同格

前後拆開後，A 是 B 或 A 如同 B；後分為主。

構成例（梵文/中文）離析
形容詞＋名詞nīlakamala（青蓮）nīlaṃ kamalam
名詞＋名詞（iva 比喻）mukhacandraḥ（面如月）mukham chandraḥ iva
名詞（A 即 B）dāna-pāramitā（布施波羅蜜）dānam eva pāramitā
地名mṛgadāva（鹿野苑）mṛga-dāva = 名為「鹿」的林子

佛典比喻複合詞：

prajñā-śastra  → 智慧劍（prajñā like śastra）
kleśa-śatru   → 煩惱賊（煩惱如賊）
dharma-ratna  → 法寶（法如珍寶）

依主釋（tatpuruṣa）——前後異格

前分限定後分，兩者是不同事物，格位不同。後分為主。

格位關係例說明
屬格依主釋nṛpasya putrah → nṛpaputraḥ（國王的兒子）前分為屬格
受格依主釋grāmagata（去了村莊的）前分為受格（+分詞）
具格依主釋—前分為具格

辨析持業釋 vs. 依主釋：

  • 持業釋：拆開後 A 和 B 同格，A 是 B 的描述
  • 依主釋：拆開後 A 和 B 不同格，A 限定 B（如屬格）

My Position

PPP 是梵文最常見的形式，能取代過去式、形容詞、名詞三種功能。閱讀佛典時，一旦看到以 -ta/-na 結尾的詞，應立即判斷：1）它修飾哪個名詞？2）字根是及物（被動）還是不及物（主動）？

**複合詞的解析（離析）**是讀佛典術語的關鍵技能。持業釋（A即B）和依主釋（A的B）是最常見的兩類。「離析」的方法：還原前分的格位（最常見的是屬格），即可得出原本的格位關係。

katama（哪一個？）是佛典問答最常見的疑問詞——「何等為四聖諦？」「katamāś catvāraḥ āryasatyāni?」

# 梵文歷史與語言分期（Sanskrit History and Periods）

梵語不是佛典的「母語」——佛陀時代明確反對使用梵語弘法。佛典使用梵語，是西元1-2世紀「梵文化」趨勢的產物，理解這個歷史分期，是閱讀佛典時辨識文獻層次的前提。

## 語言定位

- **語系**：印歐語系（Indo-European）→ 印度-伊朗語族 → 印度-雅利安語支
- **親屬語言**：古波斯語、阿維斯陀語（近親）；希臘語、拉丁語、斯拉夫語（同系）
- **共同祖源**：原始印歐語（Proto-Indo-European, PIE），1786 年由威廉·瓊斯（William Jones）提出
- **現代地位**：學術與宗教的專門語言，母語使用者極少

## 三個主要分期

### 一、吠陀梵語（Vedic Sanskrit）约 1500–500 BCE

**特徵**：
- 婆羅門（Brahmin）神職人員的專用語言
- 口傳傳統（oral tradition），為保證咒語效力，發音被嚴格保存
- 最古老文本：《梨俱吠陀》（Ṛgveda）

**與佛教的關係**：
> 佛陀時代**禁止用梵語傳佈**。佛教、耆那教等沙門思潮選用各地方言（統稱「中古印度雅利安語 / Prakrits」）弘揚教義。

→ 這意味著：早期佛教文本的「母語」是 Prakrits，不是梵語。

### 二、古典梵語（Classical Sanskrit）约 500 BCE–1000 CE

**特徵**：
- 由文法學家波你尼（Pāṇini）在其《八篇書》（Aṣṭādhyāyī, ~4th c. BCE）規範化
- 規範性文法：定義「正確的」梵語，而非描述實際用法
- 西元1-2世紀：婆羅門復興，梵語從神職語言擴展為宮廷、詩人、知識分子的語言

**與佛教的關係**：
> 西元1-2世紀，佛教為順應時代、擴大傳播，開啟**「梵文化」（Sanskritization）**風潮，大乘佛典大量以古典梵語（或接近古典梵語的形式）寫成。
>
> 但需謹記：**梵語仍是轉譯後的產物，不是佛典的原始語言。** 佛典由「中古印度雅利安諸語」結集而成。

### 三、佛教混合梵語（Buddhist Hybrid Sanskrit, BHS）

**特徵**：
- 夾雜了俗語（Prakrits）成分的梵語，形成獨特語體
- 不同文本有不同的「梵語化程度」：

| 文本 | 時期 | 語言特徵 |
|------|------|---------|
| 《大事》（Mahāvastu, MV） | 早期 | 接近俗語/Prakrits |
| 《法華經》（SDPS） | 中期 | 長行接近俗語；偈頌接近標準梵語 |
| 《入楞伽經》（LAS） | 晚期 | 接近標準/古典梵語 |

**閱讀意義**：判斷一個佛典文本的語言層次，有助於推斷其成立年代與傳播脈絡。

## 梵語對漢語的影響

許多漢語佛教詞彙是梵文的直接音譯：

| 漢語 | 梵文原詞 |
|------|---------|
| 佛 | Buddha |
| 禪 | Dhyāna |
| 菩提 | Bodhi |
| 菩薩 | Bodhisattva |
| 涅槃 | Nirvāṇa |
| 剎那 | Kṣaṇa |
| 波羅蜜 | Pāramitā |
| 阿耨多羅三藐三菩提 | Anuttarā-samyak-saṃbodhi |

梵語研究也促進了漢語音韻學的發展（反切方法源自分析梵語聲母與韻母）。

## 對佛典閱讀的核心啟示

**分期決定詮釋層次**：

```
閱讀一個佛典文本時，先問：
1. 這個文本的語言層次是什麼？（吠陀/古典/BHS/巴利）
2. 它的「梵文化程度」如何？（早期 BHS vs. 晚期近古典梵語）
3. 這個語言層次透露了什麼關於文本的成立年代和傳播背景？
```

→ 詳見 [[concepts/佛典文獻閱讀]] 的文獻層次辨識框架

## My Position

這個歷史背景對我的研究有一個重要提醒：不能把「梵語佛典」當成「佛教原始文本」——梵語化本身是一個歷史過程，是佛教為了在印度古典文化圈中傳播而做的調適。研究大乘佛典的術語時，需要意識到那個術語「被翻譯成梵語」這件事本身，可能就是研究問題的一部分。

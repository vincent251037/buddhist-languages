# 日文助動詞（Japanese Auxiliary Verbs）

日文助動詞的核心規則：**助動詞長得像哪種詞類就按那種詞類變化**。

---

## 一、助動詞分類總表（§39）

| 變化類型 | 助動詞 | 功能 |
|---------|--------|------|
| **像下一段動詞** | （ら）れる、（さ）せる、しめる | 被動、使役 |
| **像五段動詞** | たがる、である | 希望（他稱）、Be動詞（文章體）|
| **像い形容詞** | ない、たい、らしい | 否定、希望、推測 |
| **像な形容詞** | そうだ（樣態）、そうだ（傳聞）、ようだ | 樣態、傳聞、樣態 |
| **特殊變化形** | ます、た、です、ず、ざり | 美化、過去、Be動詞（敬體）、文言否定 |
| **無變化** | まい、（よ）う、ん | 否定推量、推量/勸誘、文言否定 |

---

## 二、被動・使役（§40）

### 被動（ら）れる

| 動詞類型 | 被動形 |
|---------|--------|
| 五段動詞 | V① ＋ **れる** | 
| 上下一段動詞 | V① ＋ **られる** |
| する | **される** |
| くる | **こられる** |

**被動形本身**像下一段動詞 → 加其他助動詞時，把れる/られる當下一段動詞處理。

### 使役（さ）せる

| 動詞類型 | 使役形 |
|---------|--------|
| 五段動詞 | V① ＋ **せる** |
| 上下一段動詞 | V① ＋ **させる** |

### 複合範例

```
書く① + せる（使役）① + られる（被動）① + ない（否定）② + た（過去）① + う（推量）

書か - せ - られ - なかっ - たろ - う

→ 書かせられなかったろう。
   昨天沒有被（強制）要求寫吧！
```

**還原技巧**：從後往前讀——う→たろ（推量→過去推量）→なかった（否定過去）→られ（被動）→せ（使役）→書か（書く の①変化）

## 三、様態・傳聞・推量（§43）★★★

### そうだ 兩種截然不同的意義

| 前接 | 意義 | 辨別 |
|------|------|------|
| V**②**（中止形）＋そうだ | **様態**（好像、似乎）| 動詞用②變 |
| V**③**（終止形）＋そうだ | **傳聞**（據說、聽說）| 動詞用③變 |
| い形容詞**語幹**＋そうだ | 様態 | 去掉い |
| な形容詞**語幹**＋そうだ | 様態 | 直接用語幹 |

```
見そうだ  ← 見る②中止形 → 好像要看
見るそうだ ← 見る③終止形 → 據說要看
```

### ようだ / らしい

| 助動詞 | 前接 | 意義 |
|--------|------|------|
| ようだ | V**④**（連體形）＋ようだ | 様態（好像，根據客觀觀察）|
| らしい | V**③**（終止形）＋らしい | 推測（好像，根據情報推斷）|

## 四、ます・た 特殊變化（§44）

### ます（美化/禮貌語體助動詞）

前接：動詞②（中止形）

| 変化形 | 形式 | 用例 |
|--------|------|------|
| ① | ませ/ましょ | ません（否定）；ましょう（勸誘：Let's...）|
| ② | まし | ました（過去）；まして（中止）|
| ③④ | ます | 終止/連體 |
| ⑤ | ますれ | ますれば |
| ⑥ | ませ | いらっしゃいませ（歡迎光臨）|

### た（過去助動詞）

前接：動詞・助動詞の②

⚠️ **た は②変化を持たない**（た前は必ず②変化；た自身も変化する）

| た の形式 | 変化形 | 用例 |
|---------|--------|------|
| たろ | ① | たろう（過去推量：...だったろう）|
| （なし）| ② | — |
| た | ③④ | た（句點結束/連體修飾）|
| たら | ⑤ | たら（假設語氣，後不加ば！）|

**常見複合句型**：なければならない

```
なければならない = ①＋なければ + ならない
（不這樣做不行 = 一定要做）

ない の⑤変化 = なけれ（因為ない長得像い形容詞：い→けれ）
ならない = なる①＋ない（不成/不行）

文言形：①＋ねばならぬ（同義，偶見於學術文章）
```

## 五、だ・です・である（§45）★★★

Be動詞的三種文體，**論文閱讀最重要**。

| 形式 | 文體 | 出現場合 |
|------|------|---------|
| **です** | 敬體 | 口語、對話 |
| **だ** | 常體 | 論文、學術文章（主要）|
| **である** | 文章體 | 論文、學術文章（主要）|

### だ の変化（論文必備）

| 変化形 | 形式 | 用法 |
|--------|------|------|
| ① | だろ | だろう（推量：...吧）|
| **②** | **で**（中止）/ だっ（過去）| **で：中間停止形 ★★★**；だった：過去 |
| ③ | だ | 終止（句尾）|
| ④ | **な** | 連體修飾（な形容詞語法）|
| ⑤ | **なら** | 假設語氣（なら可以...）|

**最重要：だ の②変化「で」—— 中止形** ★★★

```
阿含経とは、最も古い仏教経典集であり、釈迦の言葉を色濃く反映した...
阿含**とは** = 所謂...
**であり** = である（文章體Be動詞）の②変化（中止形）= 是...而且...

阿含とは、サンスクリット・パーリ語のアーガマの音写**で**、
「伝承された教説」という意味**である**。
この**で** = だ の②変化（中止形）= 是...（不是格助詞で！）
```

**辨別 で 是格助詞還是 Be 動詞②変化**：

| 判斷 | で の性質 | 翻譯 |
|------|---------|------|
| 名詞＋で＋動詞（方法手段）| 格助詞 | 用... |
| 名詞（=是）＋**で**、続く | だ の②変化 | 是...（而且）|

**口訣**：「學生で」翻成「用學生」或「在學生」都很奇怪 → 那就是 Be 動詞 だ の②変化「是學生...」

### である の変化（文章體）

```
である = で（だ②）+ ある（存在動詞）
→ 按 ら行五段動詞変化：ら・り（っ）・る・る・れ・れ・ろ

②であり：中止形（A はBであり、CはDである ＝ A是B且C是D）
②であった：過去（は학생であった ＝ 是學生的）
③④である：終止・連體
⑤であれ（ば）：假設
否定：では（は）ない
```

**論文範例**：

```
AはBであり、CはDである。  A是B，C是D。（用であり並列）
AはBとCである。          A是B和C。
AはBであった。           A曾是B（過去式）
AはBではない。           A不是B（否定）
```

### です の主要変化

| 変化形 | 用例 |
|--------|------|
| でした | 過去（私は学生でした）|
| でしょう | 推量（...でしょう = ...吧！）|
| ではありません | 否定（私は学生ではありません）|

## 六、文言助動詞（§46）

佛典學術文章偶見：

| 文言形 | 現代對應 | 意義 | 常見変化 |
|--------|---------|------|---------|
| **なし**（無し）| ない | 不存在、無 | ④なき（最常見）|
| **ごとし**（如し）| ～のように | 如同...一樣 | V④が＋ごとし；名詞の＋ごとし |
| **べし**（可し）| べきだ | 應該、能夠 | V③＋べし |

## 七、下樓梯模型（§47）：助動詞連接順序

助動詞必須依照特定**順序**疊加，從最靠近動詞到最末尾：

```
動詞
  ① 使役（させる・せる）
  ② 被動（られる・れる）
  ③ 否定（ない）
  ④ 過去（た）
  ⑤ 推量（う・ろう）

書かせられなかったろう。
書く ← せる ← られる ← ない ← た ← う
    使役   被動   否定   過去  推量
→ 昨天大概沒有被（強制）要求寫吧！
```

**還原方法**：從句點往前，一層一層剝開助動詞：
- 最後的推量詞 → 過去助動詞 → 否定 → 被動 → 使役 → 動詞原形

## My Position

**で（だの②変化）** 是閱讀佛典日文最關鍵的識別點。不識別這個 で，定義句的翻譯會完全出錯。凡遇到「名詞で、...」或「名詞で...という」的結構，優先判斷是 Be動詞（是...），而非格助詞（用.../在...）。

**であり / であった** 幾乎出現在每一頁佛學論文中。であり 是中止形，用來並列兩個判斷句；掌握この二形即可讀懂大多數佛學論文的論點陳述句式。

# 梵文基本句法（Sanskrit Basic Syntax）

梵語是**屈折語（inflected language）**：詞的語法功能由字尾變化（格變化、人稱變化）決定，而非詞序。語序因此相當自由，但仍有慣用習慣。

---

## 一、基本語序：S＋O＋V

梵語標準語序：**主語（S） → 賓語/受詞（O） → 謂語/動詞（V）**

```
narāḥ     nṛpam       vadanti
m.pl.N.   m.sg.Acc.   3rd.pl.pres.
人們      對著國王    說話
→ The men speak to the king.
```

**注意事項**：
1. **主謂一致**：主詞與動詞的「數」（單數/雙數/複數）必須一致
2. **梵文無冠詞**：不存在 the / a / an
3. **Be 動詞可省略**：「A 是 B」的句子，bhavati 常省略

```
aśokaḥ   nṛpaḥ   (bhavati)
阿育王   國王    [是]
→ Aśoka is the king.  （bhavati 可省）
```

---

## 二、語序彈性原則（Lesson 4）

梵語語序雖自由，但有慣例規則：

| 規則 | 說明 |
|------|------|
| 主詞在句首 | 一般位置 |
| 動詞在句尾 | 一般位置 |
| 呼格放第一或第二位置 | 呼格緊接主詞之前或之後 |
| 否定助詞（na）放動詞前 | na + 動詞 |
| 直接受詞放及物動詞前 | SOV 的 O 在 V 之前 |
| 屬格/形容詞放所修飾名詞前 | 前置修飾 |

**刻意改變語序 → 強調**：被移位的詞通常帶有強調意味。

**「間接格位」**：主格與呼格以外的所有格（受格、具格、與格、從格、屬格、位格）統稱間接格位。

---

## 三、判讀梵文句子的三步驟

讀一句梵文時，依此順序切入：

```
步驟1：找關係子句標記
        yad...tad...（前者...後者...）
        yathā...tathā...（如同...那樣...）

步驟2：找動詞
        或有動詞意味的詞類（如分詞）

步驟3：找受詞
        主動句 → 受詞在「受格」（Accusative）
        被動句 → 受詞在「主格」（Nominative）
```

---

## 四、多義詞（Polysemy）

梵語動詞詞根（√）常有多個衍生含義。處理原則：**先找基本意思，再嘗試延伸意思**。

**例：√sthā「stand」**

| 衍生含義 | 梵文例形 |
|---------|---------|
| 站立、停留 | tiṣṭhati |
| 居住（dwell） | tiṣṭhati |
| 從事於、致力於 | ─ |
| 以某狀態存在、安住 | ─ |
| 停止 | ─ |

⚠️ 注意區別：**普遍梵文（Classical Sanskrit）** vs. **佛典文獻（Buddhist Hybrid Sanskrit）** 的用法可能不同。

## 五、iti 用法（引文標記）

iti 是梵語的**引文結束標記**，功能類似引號或英文名詞子句的 that。

**位置**：說話/思考內容的**末尾**，後接表達說話、思考、知道、記得、聽到、教導等動詞。

### iti 的三種功能

| 功能 | 說明 | 中文對應 |
|------|------|---------|
| 直接引文結束 | 引號的收尾 | 「⋯⋯」|
| 名詞子句 | 相當於英文 that 子句 | 說「⋯⋯」的內容 |
| 副詞用法 | 因此、如是、如上所述 | 如是、因此 |

⚠️ iti **只能標示引文的結束**，不能標示開始——需要靠文法規則識別引文的起點。

### 例句分析

**例1：evam 配合 iti（副詞引導直接引文）**

```
narāḥ  evam  vadanti: atra  vasati  iti
人們   如此   說      他住  在這裡  [引文結束]
→ The men say thus: 'He lives here.'

evam「如此、如是」= 指向後面 iti 引文的內容
```

**例2：iti 嵌入主要子句**

```
ācāryaḥ śiṣyaḥ ca  kutra  gacchataḥ  iti  nṛpaḥ  cintayati
老師    學生   和   在哪裡  去（雙數）  [引文] 國王   思考著

→ "Where are the teacher and student going?" the king reflects.

判讀方式：
  - nṛpaḥ cintayati = 主要子句（國王思考著）
  - iti 前面是思考的內容（直接引文）
  - 動詞 gacchataḥ 是雙數 → 對應「老師和學生」兩個人
```

**例3：直述 vs. 間接敘述**

梵語習慣用**直接敘述**（保留說者原話、第一人稱）；中英文習慣**間接敘述**（名詞子句轉述、改用第三人稱）：

```
直述：atra vasāmi iti vadati
      He says: "I live here."

轉述（英文慣用）：He says that he lives here.
```

**佛典應用**：佛經常見的句式「如是我聞（evaṃ mayā śrutam）」正是 evam + iti 框架的延伸——evam 表示「如是」，後面記錄的整個法會內容以 iti 收尾。

---

## My Position

三步驟判讀法（關係子句 → 動詞 → 受格/主格）是閱讀梵文長句的第一優先框架。iti 的識別對於閱讀佛典引文結構至關重要——佛典中大量的「佛告～云：⋯⋯ 如是」正是 iti 的應用語境。Be 動詞省略在佛典中也很常見，如定義性句子「般若波羅蜜というのは～」在梵文原典往往是「prajñāpāramitā ～（bhavati 省略）」。

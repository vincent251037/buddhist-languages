# 日文基本句型與形式名詞（Japanese Sentence Patterns and Formal Nouns）

**核心**：日文句型按動詞類型（自動詞/他動詞）分叉；形式名詞是名詞化動詞句的關鍵機制；こそあど體系整合指示代名詞。

---

## 一、自動詞 vs. 他動詞（§25）

| 類型 | 英文對應 | 結構 | 例 |
|------|---------|------|----|
| **自動詞** | intransitive (Vi) | S＋V（無受詞） | ドアが閉まる（門關了）|
| **他動詞** | transitive (Vt) | S＋V＋**をに**受詞 | 鈴木さんがドアを閉める（鈴木關門）|

**重要**：日文動詞的自動詞/他動詞屬性是固定的（詞性決定，不像英文可以互換）。

---

## 二、七大基本句型（§25–27）

### 句型1：S + Vi
```
彼が行く。（He goes.）
主詞が/は＋自動詞
```

### 句型2：S + Vt + 受詞 + を/に
```
佐藤さんは本を読んだ。（S reads a book.）
主詞＋他動詞＋をorに受詞
```

**を vs. に 的受詞差異**：部分中文/英文的直接受詞，日文改用 に（間接概念）：

| に的受詞情況 | 例 | 理由 |
|------------|----|----|
| 感染疾病 | コロナに感染した | 感染的是媒介 |
| 感謝對象 | 鈴木さんに感謝する | 間接受詞 |
| 違反規則 | 交通ルールに違反する | 比較基準 |
| 回答 | 質問に答える | 對象（非直接受） |

### 句型3：A は B です/だ/である（Be動詞句）
```
私は学生です（敬體）
私は学生だ（常體）
私は学生である（文章體）
私は学生なり（古文）
```
**文言なり = だ**（對應中文古文「也」）

### 句型4：AをBと呼ぶ/言う/する
```
大乗の経論を「菩薩蔵」と呼ぶ。（把大乘稱為菩薩藏）
A＋を＋B＋と＋動詞
```

### 句型5：S + IO(に) + DO(を) + Vt（授與句）
```
彼女は本をあなたに贈る。（She gives you a book.）
```

### 句型6：被動句 ★★★
```
主動句受詞が（は）＋主動句主詞に＋動詞①＋（ら）れる

先生がこの本を書く。→ この本が先生に書かれる。
（老師寫這本書） → （這本書被老師寫）

に＝によって＝から＝より（被動句中動作發出者）
```

### 句型7：AをBとする（把A當作B）★★★
```
AをB(である)とする  → 把A當作（是）B
AをBとして         → 把A當作B（中止形，繼續後句）
AをBとすれば       → 如果把A當作B

動詞變形：とする/と言う/と思う/と考える/と呼ぶ
```

**佛學論文常見延伸**：
```
AをBとし、CをDとする＋名詞
（以A為B、以C為D的……）
```

## 三、こそあど指示體系（§50）

日文用「こそあど」代替英文 this/that/it/where，距離感由近到遠：

| 系列 | こ（近我）| そ（近你）| あ（遠離兩者）| ど（疑問）|
|------|---------|---------|------------|---------|
| **代名詞** | これ | それ | あれ | どれ |
| **地點代名詞** | ここ（此地）| そこ | あそこ | どこ（哪裡）|
| **方向代名詞** | こちら/こっち | そちら | あちら | どちら |
| **副詞** | こう（如此地）| そう | ああ | どう |
| **連體詞** | この | その | あの | どの |
| **形容詞形** | このような | そのような | あのような | どのような |
| **副詞形** | このように | そのように | あのように | どのように |

**こういう / こうした / このような** — 三者均為「這樣的」，後面接名詞：
```
こういう④＋名詞 = こうした④＋名詞 = このような④＋名詞
```

### 重要連體詞（後面一定接名詞）

| 連體詞 | 意義 | 例 |
|-------|------|---|
| **ある** | 某… | ある人（某人）；ある時（某時）|
| **いわゆる** | 所謂的 | いわゆる四諦（所謂的四諦）|
| **あらゆる** | 一切的、所有的 | あらゆる生き物（所有生物）|
| **いかなる** | 何種… | いかなる意味（何種意義）|

## 四、形式名詞（§52）★★★

形式名詞只有形式沒有內容（內容在其前面），前面的動詞句最後一個字用**第四變化（連體形）**修飾形式名詞：

| 形式名詞 | 漢字 | 意義 | 例 |
|---------|------|------|----|
| **こと** | 事 | 事情/fact | 地球が丸い**こと**は真理である |
| **もの** | 者 | 人/物/thing | 言った**もの** = 說了的那個人/東西 |
| **の** | 之 | 人/事（代替者/こと）| 教室で勉強している**の** = 正在教室讀書的（那個人/事）|
| **ところ** | 所 | 地點/時間點 | 言うところ（所言）；生ずるところ（所生）|

### 形式名詞的語法功能

```
地球が丸い④こと  は  真理である。
             ↑                ↑
        名詞子句（主詞）  述語

地球が丸いこと を 証明した。
             ↑
        名詞子句（受詞）
```

**英文對應**：`that the earth is round` = 日文的 `地球が丸いこと`

### こと vs. の の區別
| 形式名詞 | 特性 |
|---------|------|
| こと | 較正式、書面；可用於更廣的文脈 |
| の | 較口語；以前面有特定情感或具體感知內容時用 |

## 五、可能形（潛在形）（§51）

五段動詞直接變成下一段動詞，表示能力：

```
規則：五段動詞最後一個假名 う段音 → え段音 ＋ る

聞く（く→け）→ 聞ける（能聽）
読む（む→め）→ 読める（能讀）
書く（く→け）→ 書ける（能寫）
言う（う→え）→ 言える（能說）
```

**結果是下一段動詞**，按下一段動詞變化：
- 否定：聞け**ない**
- 過去：聞け**た**

---

## 六、能力表現（§54）

| 表現方式 | 型式 | 意義 |
|---------|------|------|
| **可能動詞** | 五段→下一段（§51）| 聞ける（能聽）|
| **ことができる** | V④＋ことができる | 見ることができる（可以看）|
| 否定 | ことは**できない** | 見ることはできない（不能看）|

**注意**：肯定用 が，否定通常改用 は（見ること**が**できる → 見ること**は**できない）

---

## 七、輔助動詞 ている/ていく/てくる（§53）

| 型式 | 意義 | 例 |
|------|------|----|
| V②+**ている** | 進行式（ing）或狀態 | 読んでいる（正在讀/一直讀著）|
| V②+**ていく** | 動作朝某方向延伸/消逝 | 飛んでいく（飛去）|
| V②+**てくる** | 動作朝說話者方向/持續到現在 | 飛んでくる（飛來）|

**學術論文關鍵**：
```
V②+てきた（てくる過去形）
→ 伝来してきた（傳來的）；発展してきた（發展至今的）
→ 表示某現象從過去持續到現在，在佛學論文中極常見
```

## My Position

**AをBとする / AをBとして** 是閱讀佛學論文的最核心句型之一，幾乎每頁都有。すて（とする的て形）＝ 以...為（繼續後句），せる（する的使役）以外的 とする 一定要先辨識。

**形式名詞こと** 的使用讓日文能創造類似英文 `that-clause` 的名詞子句。佛典解說文中頻繁出現「Xとはこのことである」（X 說的就是這件事）這類定義句型。

**てきた** 在佛學研究論文中意義深遠——用來描述傳統/觀念「歷史上一直延續到現在」的狀態，是歷史敘述句的標誌性句型。
